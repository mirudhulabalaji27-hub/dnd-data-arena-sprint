Slide 05 · Methodology
How We Built It
Step 01
📥
Data Acquired
Downloaded Spotify User Behavior dataset from Kaggle. 520 rows × 20 columns in XLSX format.
Step 02
🧹
Cleaned in Colab
Handled nulls, removed duplicates, split multi-select columns, standardized all column names.

📊
